What Is My IP Address (IPv4 & IPv6)?

A public IP address is the unique identifier assigned to your device when you connect to the Internet. With the What’s My IP tool, you can instantly view your public IP (IPv4 or IPv6) along with other useful connection details.

Whenever you use a new Wi-Fi network or switch devices, your IP address may change. Knowing your IP is helpful for tasks such as network configuration, troubleshooting, and traffic analysis.

Information provided by the tool

  • Approximate location: region associated with the IP (not GPS).
  • ISP: Internet Service Provider.
  • Operating system: Windows, macOS, Linux, Android, etc.
  • Browser: Chrome, Firefox, Safari, entre outros.
  • Screen resolution: device display size.
  • Latitude and longitude: approximate IP coordinates.

Note: IP geolocation is not 100% accurate and only indicates the area where the IP address is registered.

FAQs

What is a private IP address and how can you find it?

A private IP address is used to identify devices within a local network. When a device communicates with other devices or with the network router, it uses a private IP. To communicate with the Internet, a public IP address is assigned.

You can find your device’s private IP by opening the command prompt (CMD), typing ipconfig, and pressing Enter.

What is a VPN and how does it relate to your public IP?

VPN stands for Virtual Private Network. When you connect to a VPN, your public IP address is replaced with another one, typically associated with a different country.

VPNs are used to access region-restricted content and for privacy purposes, as they hide your real public IP during browsing.

Can someone track my exact location using my IP address?

No. An IP address cannot identify a person’s exact location. It only provides an approximate location, such as a city or region, never specific details like street name or house number.
