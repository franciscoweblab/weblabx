How to check MX records

  1. Enter the domain.
  2. Select the DNS server.
  3. Click "Check MX Record".
  4. Review the mail servers.

Checking MX records ensures your domain can receive emails correctly and helps prevent delivery failures.

About MX Lookup

MX Lookup is a free online tool that lets you query the MX (Mail Exchange) records of any domain in real time.

These records define which mail servers are authorized to receive messages for a domain. Incorrect configurations can cause delivery failures.

FAQ

What is an MX record?

An MX record defines which servers receive emails for a domain.

What does priority mean?

Lower numbers indicate higher priority mail servers.
