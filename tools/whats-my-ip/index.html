What Is My IP Address (IPv4 & IPv6)?

O endereço IP público é o identificador único atribuído ao teu dispositivo quando te ligas à Internet. Com a ferramenta What’s My IP, podes ver instantaneamente o teu IP público (IPv4 ou IPv6) e outras informações úteis sobre a tua ligação.

Sempre que usas uma nova rede Wi-Fi ou mudas de dispositivo, o endereço IP pode ser diferente. Saber qual é o teu IP ajuda em tarefas como configuração de redes, resolução de problemas e análise de tráfego.

Informações apresentadas pela ferramenta

  • Localização aproximada: região associada ao IP (não é GPS).
  • ISP: fornecedor do serviço de Internet.
  • Sistema operativo: Windows, macOS, Linux, Android, etc.
  • Navegador: Chrome, Firefox, Safari, entre outros.
  • Resolução do ecrã: tamanho do ecrã do dispositivo.
  • Latitude e longitude: coordenadas aproximadas do IP.

Nota: a localização do IP não é 100% precisa e indica apenas a zona onde o endereço IP está registado.

FAQs

O que é um endereço IP privado e como o podes encontrar?

Um endereço IP privado é utilizado para identificar dispositivos dentro de uma rede local. Quando um dispositivo comunica com outros dispositivos ou com o router da rede, utiliza um IP privado. Para comunicar com a Internet, é atribuído um endereço IP público.

Podes encontrar o IP privado do teu dispositivo abrindo a linha de comandos (CMD), escrevendo ipconfig e pressionando Enter.

O que é uma VPN e que relação tem com o IP público?

VPN significa Virtual Private Network. Ao ligares-te a uma VPN, o teu endereço IP público é alterado para outro, normalmente associado a um país diferente.

As VPNs são usadas para aceder a conteúdos bloqueados por região e também por motivos de privacidade, uma vez que escondem o teu IP público real durante a navegação.

É possível alguém rastrear a minha localização exata através do meu endereço IP?

Não. Um endereço IP não permite identificar a localização exata de uma pessoa. Apenas fornece uma localização aproximada, como cidade ou região, e nunca dados específicos como rua ou número da casa.
